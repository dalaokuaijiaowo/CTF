由题目得16进制转换
使用密码工具箱
https://box.deepzz.com/
将字符串每两个字符分成一组（每组代表一个十六进制字节）：

4c 69 74 43 54 46 7b 74 61 69 31 31 31 63 6f 6f 6c 6c 61 61 61 21 7d
将每个十六进制字节转换为对应的ASCII字符：

4c -> L
69 -> i
74 -> t
43 -> C
54 -> T
46 -> F
7b -> {
74 -> t
61 -> a
69 -> i
31 -> 1
31 -> 1
31 -> 1
63 -> c
6f -> o
6f -> o
6c -> l
6c -> l
61 -> a
61 -> a
61 -> a
21 -> !
7d -> }
将所有字符组合起来，得到解码后的字符串：

LitCTF{tai111coollaaa!}
